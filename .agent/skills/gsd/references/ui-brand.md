# UI Patterns for Telegram

## Stage Indicators

**Use:**
```
 **GSD → Stage Name**
```

**Not:**
```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
 GSD ► STAGE NAME
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## Menus

**Use:**
```
**Ready to proceed?**

1. Create PROJECT.md
2. Keep exploring

Reply with 1 or 2
```

## Progress

**Use:**
```
 **Phase 1 complete**

Created:
• `.planning/PROJECT.md`
• `.planning/ROADMAP.md`

**Next:** `/gsd plan-phase 1`
```

## Keep It Clean
- Telegram markdown: **bold**, *italic*, `code`
- No ASCII art boxes
- Emojis: minimal ( only)
- Conversational tone
